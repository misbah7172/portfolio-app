## YelpCamp: Discover Your Next Adventure

A feature-rich platform for camping enthusiasts to discover, review, and share their favorite campgrounds. Includes location mapping, user reviews, and detailed campground information.

### Stack

- [Node.js](https://nodejs.org) - Backend runtime environment
- [Express](https://expressjs.com) - Web application framework
- [MongoDB](https://mongodb.com) - NoSQL database
- [Mapbox](https://mapbox.com) - Interactive mapping platform
