## Notes App: Streamlined Note Taking

A clean and efficient note-taking application designed for simplicity and functionality. Features include markdown support, categories, search functionality, and automatic saving.

### Stack

- [React](https://reactjs.org) - Frontend library for user interfaces
- [Firebase](https://firebase.google.com) - Backend and real-time database
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework
- [React Markdown](https://github.com/remarkjs/react-markdown) - Markdown rendering
