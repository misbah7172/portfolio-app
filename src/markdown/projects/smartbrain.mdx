## SmartBrain: AI-Powered Face Detection

An intelligent application that leverages machine learning to detect and analyze faces in images. Features include user authentication, image URL processing, and face location highlighting.

### Stack

- [React](https://reactjs.org) - Frontend framework for user interfaces
- [Node.js](https://nodejs.org) - Backend runtime environment
- [PostgreSQL](https://postgresql.org) - Relational database
- [Clarifai API](https://clarifai.com) - AI-powered image recognition
