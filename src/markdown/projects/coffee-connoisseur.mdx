## Coffee Connoisseur: Discover Your Perfect Cup

An interactive platform for coffee lovers to discover, rate, and share their favorite local coffee shops. Features include geolocation-based recommendations, user ratings, and detailed shop information.

### Stack

- [Next.js](https://nextjs.org) - React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework for modern web development
- [Foursquare API](https://developer.foursquare.com) - Location data and business information
- [Airtable](https://airtable.com) - Flexible database for storing user preferences and ratings
